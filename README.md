# MPEB Electricity Bill Fetcher

A simple web application to fetch electricity bill details for customers of **M.P. Pashchim Kshetra Vidyut Vitaran Co. Ltd. (MPEB)**.  
Users can enter their **IVRS (customer) number** to view the latest bill details directly.

## Features
- Quick bill lookup using the IVRS number
- Simple and clean user interface
- No login or signup required
- Fetches latest bill information in real-time (based on publicly accessible data)

---

## How to Use
1. Open the website.
2. Enter your **IVRS (Customer) Number**.
3. Click **View Bill**.
4. View your latest electricity bill details.
